# botcoin

Bot pour les palmipèdes

Commandes et sondage

charger les nouvelles commandes : `node .\deploy-commands.js`
lancer : `node index.js`


Ajouter le bot : 
- Dans le portail développeur de Discord
    - OAuth2
        - cocher BOT
        - Choisir les droits
        - Copier l'URL
- Coller l'URL dans un navigateur
